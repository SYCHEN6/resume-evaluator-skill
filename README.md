# Resume Evaluator Skill

一个基于 Claude Code Skill 系统的简历评估工具，通过分析职位描述（JD），对候选人简历进行多维度打分、差距分析，并给出带有原文引用和改写示例的具体修改指导。

## 功能特性

- **多维度评分**：从技能匹配、经验相关性、学历背景、关键词覆盖、简历质量五个维度综合评分（满分100）
- **差距分析**：逐条对照 JD 找出候选人的真实能力缺口，区分"经历不足"和"真实能力缺口"
- **具体修改指导**：每个修改点提供 问题说明 → 原文引用 → 建议改写 的完整对照，可直接复制使用
- **ATS 关键词优化**：分析 JD 中候选人简历缺失的 ATS 高频词，并给出插入位置建议
- **生成修改版简历**：用户确认后，可基于修改建议生成完整的修改版简历（Markdown 格式，含修改说明）
- **支持中英文**：支持中文简历、英文简历、中文 JD、英文 JD 的任意组合，报告始终输出中文
- **支持 PDF 简历**：Claude 可直接读取 PDF 文件，无需手动复制文本

## 适用场景

- 社会招聘（3年以上工作经验），不适用于校招场景
- 目标岗位：Java 开发、Python 开发、Agent 应用工程师、大模型应用工程师等技术开发岗位
- 企业类型：私企、国企、外企均适用，外企英文 JD 场景下可保留英文关键词供 ATS 使用

## 使用方法

### 安装

将 `SKILL.md` 文件放置到 Claude Code 的 skills 目录，或通过插件系统加载。

### 触发方式

在 Claude Code 中，满足以下任意条件时会自动触发此 Skill：

- 用户提供 JD + 简历，询问匹配度或优化建议
- 用户说"帮我看看简历"、"怎么改简历"、"这个 JD 我符合吗"
- 用户上传简历文件（PDF/文本）并附上职位描述

### 使用示例

```
用户：帮我评估一下我的简历和这个JD的匹配程度，给出优化建议。

【JD内容粘贴在这里】

【简历内容粘贴在这里，或上传PDF文件】
```

也支持先提供 JD，Skill 会自动询问简历。

## 输出格式

评估报告包含以下部分：

```
# 简历评估报告

## 综合评分：XX / 100
（5维度评分表格）

## 亮点分析
（2-4条 与JD匹配的竞争优势）

## 差距分析
（逐条列出JD要求但简历缺失的内容）

## 简历修改指导
### 🔴 必须修改（影响是否过简历筛选）
### 🟡 建议优化（提升说服力）
### 🟢 关键词补充建议

## 总结与行动优先级
```

报告结尾会询问是否需要生成**修改版完整简历**（Markdown 格式，含修改说明注释块）。

## 设计原则

- **不虚构经历**：修改建议只针对简历中真实存在的经历进行重新表达，JD 中有但候选人未接触的技术只列为差距，不建议候选人写入简历
- **诚实评分**：45分就是45分，不粉饰，差距是可修复的，不是致命的
- **ATS 优先**：关键词匹配往往是第一道过滤，先确保过 ATS 再优化人工阅读体验
- **转型友好**：对转型求职者会明确区分"包装不足"和"能力缺口"，帮助判断这份 JD 是否值得投

## 测试用例

`evals/evals.json` 包含4个测试场景：

| ID | 场景 | 说明 |
|----|------|------|
| 1 | Java高级工程师 → 国有银行 | 测试技术栈缺口识别（Spring Cloud）和国企场景 |
| 2 | Python工程师转型 → Agent工程师 | 测试转型场景处理和个人项目价值评估 |
| 3 | LLM工程师 → 外企英文JD | 测试英文JD中文报告、ATS关键词保留英文 |
| 4 | Java工程师 → 国有银行（请求生成修改版简历） | 测试完整两步流程（评估 + 生成修改版） |
